# Contexto para Claude Code

**LEER PRIMERO**: El archivo **AGENTS.md** en la raíz del proyecto es la única fuente de verdad (Source of Truth) para las reglas de arquitectura, estilo de código y comportamiento de la IA.

**Instrucción Obligatoria**:

1. Antes de realizar cualquier tarea, lee detenidamente `AGENTS.md`.
2. Sigue estrictamente las directivas universales y consulta los archivos `SKILL.md` dentro de `.agent/skills/` según la tarea a realizar (Frontend, DB, i18n, etc.).

No dupliques información. Mantén el contexto limpio y enfocado.
